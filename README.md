
Este aplicativo foi feito por mim, Silvio Aurich Filho, destinado principalmente para o pessoal das famílias a mim relacionadas.
Foi feito com muito carinho, lembrando dos familiares, e cuidando para não causar desconfortos, como forçar a lembrança dos aniversários 
de pessoas já falecidas, permitir inclusão daqueles que fazem parte da família por conta de casamentos desfeitos, adoções, etc.
Ao final, coloquei uma lista de aniversários que imagino ser útil para quem utiliza um aplicativo assim.

**Descartei a colocação dos dados na nuvem** para não expor nomes, dados e parentescos dos participantes. Assim, cada um poderá montar a 
sua rede de família, particular, apagando registros, substitundo por outros mais relacionados a si e os mais próximos.
A idéia é que o aplicativo e os dados sejam passados entre pessoas da família, e que troquem informações através do arquivo <arvore(*).json> 
que normalmente ficará salvo na pasta Downloads dos dispositivos (celular ou computador).
Mesmo tendo utilizado técnicas de auxílio de Inteligência Artificial, o projeto deu semanas de trabalho com aqueles "faz-e-refaz". Espero que 
seja útil e agradável a todos.

abraços, Silvio.
